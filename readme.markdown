# SV Angel Website User Guide

# How To

## Compile Sass to CSS

    $ compass watch

## Build blog

    $ jekyll --pygments --safe

## View site

    $ cd _site
    $ python -m SimpleHTTPServer

or

    $ jekyll --pygments --safe --server
